My First Site — Quick Start

This is a minimal static site (HTML/CSS/JS) scaffold.

Run locally:

Windows / macOS / Linux

```bash
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

Deploy to GitHub Pages:

- Initialize a git repo, commit, push to GitHub.
- In repository settings enable GitHub Pages from the `main` branch (or use `gh-pages` branch).

Next steps:

- Personalize content in `index.html`.
- Add images and more pages.
